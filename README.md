# Airbnb Asheville Market Analysis

## Objective
Evaluate Asheville as a potential Airbnb expansion market by analyzing pricing trends,
neighborhood activity, host behavior, and demand patterns.

## Data
- listings.csv
- reviews.csv
- neighbourhoods.csv

## Tools
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Entire homes dominate listings
- Demand peaks seasonally
- Certain neighborhoods show high demand with moderate competition

## Conclusion
Asheville presents strong potential for targeted Airbnb expansion.
